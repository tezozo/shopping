## Commands

**Backend** (run from project root):
```bash
./gradlew bootRun          # start Spring Boot on :8080
./gradlew build            # compile + test
./gradlew test             # run tests
./gradlew test --tests "com.highspring.shopping.SomeTest"  # single test
```

**Frontend** (run from `frontend/`):
```bash
npm install    # first time only
npm run dev    # Vite dev server on :5173 (proxies /api → :8080)
npm run build  # production build
```

H2 console available at http://localhost:8080/h2-console (JDBC URL: `jdbc:h2:mem:shoppingdb`, user: `sa`, no password).

## Architecture

### Backend — Spring Boot 3 / Java 21 / H2

Standard layered layout at the project root (`src/main/java/com/highspring/shopping/`):

| Package | Purpose |
|---|---|
| `entity/` | JPA entities — direct DB representation |
| `repository/` | Spring Data JPA interfaces |
| `service/` | Business logic (`ShoppingCartService`) |
| `controller/` | REST controllers, all prefixed `/api/` |
| `dto/` | Request records (`CreateShoppingCartRequest`, `UpdateShoppingCartRequest`, `ShoppingCartItemRequest`) |
| `config/` | `CorsConfig` (allows `:5173`), `DataInitializer` (seeds 4 items on startup) |

**Entity relationships:**
- `ShoppingCart` → `ShoppingCartItem[]` (OneToMany, cascade ALL, orphanRemoval, EAGER)
- `ShoppingCartItem` → `Item` (ManyToOne, EAGER)
- `Item` → `Set<Tax>`, `Set<Category>` (ManyToMany, EAGER, join tables `item_taxes` / `item_categories`)
- `CategoryDiscount` links `Category` ↔ `Discount`
- `ShoppingCart.id` is a UUID generated by Hibernate; all other PKs are `IDENTITY` longs

Soft delete: `ShoppingCart.deleted` flag + `deletedOn` timestamp; `findByOwnerAndDeletedFalse` filters these out.

Schema is `create-drop` — recreated on every restart; `DataInitializer` re-seeds on each boot.

### Frontend — React 18 / Vite

Located in `frontend/`. Vite proxies `/api/*` to the backend so no CORS issues during development.

| File | Purpose |
|---|---|
| `src/App.jsx` | Root — owns `owner` state (persisted in `localStorage`), renders routes |
| `src/components/OwnerInput.jsx` | Username gate shown before any cart UI |
| `src/components/CartList.jsx` | Lists/creates/deletes carts for the current owner |
| `src/components/CartDetail.jsx` | View and edit items in a single cart; computes line totals including taxes |
| `src/api/carts.js` | Axios wrappers for all 5 cart endpoints |
| `src/api/items.js` | Axios wrapper for `GET /api/items` |
| `src/api/client.js` | Shared Axios instance (`baseURL=/api`) |
